前言

JS对象同其他语言一样，都是一种无序的集合数据类型，由若干键值对构成，JS中所有的事物都是对象，包括字符串，数值，数组，函数等。常用于存储信息。

键值对就是所谓的属性和方法。

属性：描述事物的特征，在对象中用属性来表示。通常名词表示，包括字符串，数组，数值，布尔值等。

方法：描述事物的方法，在对象中用方法来表示。通常是一个函数。

声明一个简单的学生对象：

 var Student= {
     name: '张三',
     age: 150,
     sayHi: function(){
        console.log('Hi');
     }
 }

1. 创建对象

1.1 字面量

上面的例子就是采用字面量声明的对象，字面量即直接采用对象符号{}声明。直接将描述事物的所有的对象写在{}内，并赋值给被描述的事物

1.2 new关键字

// 先声明一个对象
var Student = new Object()
// 将属性和方法依次添加给对象
Student.name = '张三'; //注意分号
Student.age = 15;
Student.sayHi = function () {
    console.log('Hi');
}

实际上new就是用来创建构造函数的实例，上面的Object和下文中的Star都是一个构造函数，Object是ES内置的构造函数，Star是自定义的构造函数，构造函数后面跟括号调用该函数，返回的就是一个对象，就是该构造函数的实例对象，对象中包含属性合方法

所有对象都有构造函数，可直接通过对象的constructor属性访问到

1.3 构造函数

什么是构造函数：

个人认为的构造函数就是相当于一个对象集合，由于上面的声明的方式若需要创建多个对象，便重复了很多重复赋值的操作，而构造函数的作用就是将相同的代码下写在一个函数中，只需要在创建特定对象的时候将特定的参数传进去即可创建一个单独的对象

例:

// 创建构造函数
function Star(uname, age, sex) {
    this.name = uname;
    this.age = age;
    this.sex = sex;
    this.sing = function (sang) {
        console.log(sang);
    }
}
// 利用new声明对象  传入对应的参数
var p1 = new Star('zs', 18, '男'); //方法的参数需要在调用的时候传入

2. 访问对象

2.1 访问属性

  1. 通过.调用属性 Object.attribute

例:

 var Student= {
     name: '张三',
     age: 150,
     sayHi: function(){
        console.log('Hi');
     }
 }
 console.log(Student.name) //张三
 console.log(Student.age) //150
  1. 通过[]调用 Object[‘attribute’]

该方法适用于当属性名比较复杂时调用

var Student = new Object()
Student.name = '张三'; //注意分号
Student.age = 15;
Student.sayHi = function () {
    console.log('Hi');
}
console.log(Student['name'])

2.1 访问方法

访问方法需要单独调用，调用格式： Object.function()

例：

 var Student= {
     name: '张三',
     age: 150,
     sayHi: function(){
        console.log('Hi');
     }
 }
 Student.sayHi();

访问构造函数方法：

function Star(uname, age, sex) {
    this.name = uname;
    this.age = age;
    this.sex = sex;
    this.sing = function (sang) {
        console.log(sang);
    }
}
 var p1 = new Star('zs', 18, '男');
//  需要传参
 p1.sing('xxxx'); //xxxx
