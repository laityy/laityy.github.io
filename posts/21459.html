JavaScript事件

前言

事件是HTML DOM的核心之一，因而也可称为HTML事件，HTML事件是发生在HTML元素上的事情，在HTML页面中使用JavaScript时，JS可触发这些事件，事件可以是用户行为，也可以是浏览器行为，用户行为指的是用户通过输入设备输入的一定指令所触发的事件，浏览器行为是页面完成某项任务后自动执行的事件。

通过事件可动态改变HTML页面中的内容，属性，样式等，从而实现页面的基本交互效果。

事件三要素

  1. 事件源： 通俗来说就是需要让哪个元素触发事件就需要将哪个事件作为事件源
  2. 事件类型：即让事件源通过哪一种行为触发，常见包括鼠标点击，键盘按下等
  3. 事件处理程序：给事件源绑定事件类型后，触发该事件类型后所需要完成的工作，即对事件作出的响应，通常为一个JS函数赋值的方法

例：

<html>
<head> 
<meta charset="utf-8"> 
</head>
<body>
<button id="btn">点击</button>
<script>
// 获取button事件源
 var btn = document.getElementById('btn');
//  为button绑定点击事件
 btn.onclick= function(){
    // function就是事件处理函数，需要完成的工作都放在function内
     alert('弹出了一个对话框')
 }
</script>
</body>
</html>

1.获取事件源

1.1 getElementById():

该方法用于通过id从dom中获取元素，类似的还有：getElementsByTagName() getElementsByName() getElementsByClassName()需要注意的是id是唯一的，但标签名和类名不是唯一的，所以通过标签名获取可以获得多个具有相同类名的元素，getElementsByTagName() getElementsByClassName()方法返回 HTMLCollection 对象,需要与getElementsByName()返回的NodeList相区分

<html>
<head>
<meta charset="utf-8">
</head>
<body>
<p class='p' name='demo'>Hello World!1</p>
<p class='p' name='demo'>Hello World!2</p>
<p class='p' name='demo'>Hello World!3</p>
<script>
var Collection1 = document.getElementsByTagName("p");
var Collection2 = document.getElementsByClassName("p");
var Collection3 = document.getElementsByName('demo');
var Collection4 = document.querySelectorAll('.p');
console.log(Collection1) //HTMLCollection(3) [p.p, p.p, p.p, demo: p.p]
console.log(Collection2) //HTMLCollection(3) [p.p, p.p, p.p, demo: p.p]
console.log(Collection3) //NodeList(3) [p.p, p.p, p.p]
console.log(Collection4) //NodeList(3) [p.p, p.p, p.p]
</script>
</body>
</html>

2.1 querySelector():

querySelector() 方法返回文档中匹配指定 CSS 选择器的一个元素，若返回多个元素使用使用querySelectorAll()，返回 NodeList 对象。

2.2 NodeList和HTMLCollection区别

HTMLCollection是HTML元素的集合，
NodeList是一个文档节点的集合。

相同点：

  1. 两者都与数组对象类似，可以使用索引(0,1,2…)来获取元素，且都有length属性

不同点:

  1. HTMLCollection元素可以通过name，id或索引来获取。 NodeList只能通过索引来获取
  2. NodeList是静态集合，HTMLCollection是一个动态集合。其中动态集合和静态集合的最大区别在于：动态集合指的就是元素集合会随着DOM树元素的增加而增加，减少而减少；静态集合则不会受DOM树元素变化的影响。
<html>
<head>
<meta charset="utf-8">
</head>
<body>
<div>
    <p>Hello World!</p>
    <p>DOM 是非常有用的!</p>
</div>
<script>
    x=document.getElementsByTagName("p");  // 注意该代码要写在创建元素前面才体现出二者区别
    var p1 = document.createElement('p')
    var div = document.querySelector('div');
    p1.innerHTML='p1'
    div.appendChild(p1)
    console.log(x);
    console.log(x.length);  // 返回3 若使用querySelectorAll返回2  
</script>
</body>
</html>

2.常见事件类型

2.1 鼠标事件

  1. onclick: 鼠标左键点击触发

  2. onmouseover,onmouseout: 鼠标经过/离开(指定DOM元素)触发

<html>
<head> 
<meta charset="utf-8"> 
</head>
<body>
<div id='demo' style="width: 200px;height: 200px; background-color: red;"></div>
<script>
var demo = document.querySelector("#demo");
demo.onmouseover=function(){
    this.style.backgroundColor='blue';
}
demo.onmouseout=function(){
    this.style.backgroundColor='red';
}
</script>
</body>
</html>
  1. onmouseup,onmousedown: 鼠标弹起/按下(指定DOM元素)触发
<html>
<head> 
<meta charset="utf-8"> 
</head>
<body>
<div id='demo' style="width: 200px;height: 200px; background-color: red;"></div>
<script>
var demo = document.querySelector("#demo");
// demo被点击时，触发 onmousedown 事件
demo.onmousedown=function(){
    this.style.backgroundColor='blue';
}
// 当鼠标按钮被松开时，会触发 onmouseup 事件
demo.onmouseup=function(){
    this.style.backgroundColor='pink';
}
// 最后，当鼠标点击完成时，触发 onclick 事件
demo.onclick=function(){
    this.style,backgroundColor='black';
}
</script>
</body>
</html>
  1. onmousemove: 鼠标在DOM元素中移动时就触发
<html>
<head> 
<meta charset="utf-8"> 
</head>
<body>
<div id='demo' style="width: 200px;height: 200px; background-color: red;"></div>
<script>
var demo = document.querySelector("#demo");
// 鼠标在div内移动时会不断触发事件打印输出 ‘鼠标移动了’
 demo.onmousemove=function(){
     console.log('鼠标移动了')
 }
</script>
</body>
</html>
  1. onfocus,onblur: 目标表单得到/失去焦点时触发
<html>
<head>
<meta charset="utf-8">
</head>
<head>
<script>
function Function1(x){
    x.style.background="yellow";
}
function Function2(x){
    x.style.backgroundColor="";
    }
</script>
</head>
<body>
<!-- 当输入框获取焦点时，修改背景色（background-color属性）为yellow，当输入框失去焦点时触发Function2，将背景色去掉 -->
<input type="text" onfocus="Function1(this)" onblur="Function2(this)">
</body>
</html>

2.2 键盘事件

  1. onkeydown: 某个键盘按键被按下
<html>
<head>
<meta charset="utf-8">
<script>
function myFunction(){
    alert("你在输入栏内按下一个键");
}
</script>
</head>
<body>
<!-- 在输入框内按下一个按键是函数被触发，先弹出警示框，后写入字符-->
<input type="text" onkeydown="myFunction()">
</body>
</html>
  1. onkeypress: 某个键盘按键被按下并松开，类似onclick

在所有浏览器中 onkeypress 事件不是适用于所有按键(如： ALT, CTRL, SHIFT, ESC)。监听用户是否按下按键应使用 onkeydown 事件,所有浏览器都支持 onkeydown 事件。
onkeypress相对使用较少，功能与onkeydown类似

<html>
<head>
<meta charset="utf-8">
<script>
function myFunction(){
    alert("你在输入框内按下一个按键");
}
</script>
</head>
<body>
<!-- 在输入框内按下一个按键时函数被触发 -->
<input type="text" onkeypress="myFunction()">
</body>
</html>
  1. onkeyup: 某个键盘按键被松开
<html>
<head>
<meta charset="utf-8">
<script>
function myFunction(){
    var x=document.getElementById("fname");
    x.value=x.value.toUpperCase();
}
</script>
</head>
<body>
<!-- 在输入字段释放一个按键时触发函数。函数将字符转换为大写 -->
<input type="text" id="fname" onkeyup="myFunction()">
</body>
</html>

3.事件处理程序

3.1 事件处理程序介绍

当特定的操作发生时，就会发生事件。把事件处理程序关联到事件发生时要处理的代码上，就提供了在事件发生时捕获事件并执行相应代码的方式。就像设置闹钟，当某个事件发生时，触动事件处理程序。
